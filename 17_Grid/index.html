Topic 20 – performance & accessibility notes

Keep grids reasonable, test keyboard order & screen readers.

- Avoid huge explicit grids
- Keep DOM order logical; CSS should not hide important content order
- Use aria roles where needed (e.g., role="grid" for interactive grids)
- Test with reduced motion and large-font settings
